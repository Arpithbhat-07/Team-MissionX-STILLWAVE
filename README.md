🌊 STILLWAVE
Pop Stress. Feel Light.

STILLWAVE is a frictionless, touch-based stress relief web application designed to reduce cognitive load and promote emotional regulation through interactive micro-interventions.

Instead of journaling, typing, or navigating complex interfaces, users simply touch, hold, swipe, and breathe.

🧠 Problem Statement

Traditional mental health applications demand active cognitive effort — filling forms, writing thoughts, tracking moods.

But when someone is anxious, overwhelmed, or mentally fatigued:

Thinking is the last thing they want to do.

STILLWAVE removes cognitive friction by offering intuitive, body-first regulation tools powered by motion, sound, and haptics.

✨ Core Features
🌊 1. Guided Breathing Orb

Adaptive breathing patterns (4-6-2, 4-7-8, Reset Breath)

Dynamic orb scaling animation

Custom-built haptic vibration patterns for:

Inhale (rising pulses)

Hold (steady throb)

Exhale (falling pulses)

Visual progress indicators

🌀 2. Swipe-to-Release Thoughts

Contextual thought cards (low / medium / high stress)

Gesture-based drag & fling interaction

Visual particle “release” effects

Progress tracking dots

Subtle haptic feedback on release

🎹 3. Interactive Piano Therapy

Fully functional WebAudio-based synthesizer

Custom-built reverb and envelope shaping

Falling note game mode

Built-in song presets:

Moonlight

River Flows in You

Canon in D

Free Play

Keyboard + touch support

Hit detection + ripple feedback

Score tracking

🫧 4. Squeeze Stress Ball

Canvas-based real-time deformation

Pressure intensity meter

Stress release counter

Particle feedback animation

Haptic response

🎨 Immersive Background System

Canvas-based animated particle field

Floating interactive bubbles

Radial gradient mood lighting

Noise overlay for texture depth

🛠 Tech Stack

HTML5

CSS3 (Advanced animations & gradients)

Vanilla JavaScript

WebAudio API

Canvas API

Navigator Vibration API

No external frameworks.
Fully custom-built interaction engine.

📱 Design Philosophy

STILLWAVE follows a Zero Cognitive Load Model:

No signups

No typing

No instructions

Just interaction

The user regulates through touch and sensory feedback, not logic.
